Acerca de

Este trabajo y su historia

  • 01/08/2016 - 01/09/2016

    Nuestros Inicios

    Este proyecto parte de una necesidad para una ficticia empresa en la materia Ingenieria Web, la cual requiere de una aplicacion web para su sistema de PQRS. Nuestro compromiso es ganar con una alta nota y para ellos diseñamos esta aplicación de calidad

  • 01/09/2016 - 01/10/2016

    Nace un proyecto

    El primer paso para el cumplimiento de nuestro objetivo fue la toma de requisitos, los cuales implican aquellas funcionalidades y caracteristicas principales de nuestra aplicación

  • 01/10/2016 - 01/11/2016

    Inicios del Desarrollo

    Para el desarrollo de la aplicación web se hicieron uso de tecnologias de punta y del lenguaje de programación Java

  • 01/11/2016 - 28/11/2016

    Culminación del proyecto

    Para desarrollar la fase final de la aplicación web se usó el framework AngularJS 1.5, que en conjunto con otras tecnologias como Javascript, html 5, css 3 y bootstrap, ayudaron a que el proyecto fuese todo un exito y como resultado usted esta viendo esta hermosa aplicación
